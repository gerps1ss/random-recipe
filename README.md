# Случайный рецепт из того, что есть

> Веб-сайт для подбора случайного рецепта на основе доступных ингредиентов

## О проекте
Проект выполнен в рамках проектной практики студентами Московского Политеха (группа 253-331).

---

## Цель проекта

Разработать сайт с простым и понятным интерфейсом, позволяющий пользователю получать случайный рецепт по списку имеющихся ингредиентов.

---

## Задачи проекта

- Изучить предметную область и определить функциональные требования
- Спроектировать структуру страниц сайта
- Реализовать пользовательский интерфейс
- Проверить корректность работы сайта
- Разместить проект в GitHub и оформить историю изменений
- Подготовить отчет по выполненной работе

## Структура сайта

- `index.html` — главная страница
- `about.html` — о проекте
- `recipes.html` — рецепты
