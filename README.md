## Вариативная часть задания

Тема проекта: **«Случайный рецепт из того, что есть»**.

В рамках вариативной части был разработан многостраничный веб-сайт, позволяющий пользователям находить рецепты на основе имеющихся ингредиентов.

Основные возможности проекта:

* поиск рецептов по ингредиентам;
* фильтрация и сортировка рецептов;
* просмотр информации о проекте;
* просмотр информации об участниках;
* журнал выполнения проекта;
* раздел полезных ресурсов.
