# Документация проекта «Случайный рецепт из того, что есть»

## Описание проекта

Проект представляет собой многостраничный веб-сайт, позволяющий пользователям находить рецепты на основе имеющихся ингредиентов.

Основная цель проекта — помочь пользователям быстро подобрать рецепт из доступных продуктов, сократить пищевые отходы и упростить процесс приготовления пищи.

## Используемые технологии

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* Markdown

## Этапы выполнения проекта

### 1. Подготовительный этап

* Создан репозиторий GitHub.
* Изучены требования проектной практики.
